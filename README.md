# Portfolio — Satyam Dubey

Personal portfolio website built with React + Vite + Tailwind CSS.

## Stack

- React 19, Vite 8
- Tailwind CSS v4
- Framer Motion
- React Router v7

## Pages

- `/` — Work (project grid)
- `/project/:id` — Case study detail
- `/resume` — Resume
- `/hobby` — Moonlight (digital zine)

## Dev

```bash
npm install
npm run dev
```
